# SpaceX Falcon 9 First Stage Landing Prediction

## Project Overview
This project is part of the IBM Data Science Professional Certificate Capstone.  
The objective is to analyze SpaceX Falcon 9 launch data and predict the success of first-stage landings using data science techniques.

## Project Structure
- Data Collection and Wrangling
- Exploratory Data Analysis (EDA)
- SQL-based Analysis
- Interactive Visual Analytics
- Machine Learning Classification Models
- Interactive Dashboard using Plotly Dash

## Tools & Technologies
- Python
- Pandas, NumPy
- SQL (SQLite)
- Matplotlib, Seaborn, Plotly
- Plotly Dash
- Scikit-learn

## Key Outcomes
- Identified launch sites with highest success rates
- Analyzed payload impact on landing success
- Built classification models to predict landing success
- Developed an interactive dashboard for visual insights
